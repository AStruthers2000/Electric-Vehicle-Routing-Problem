Generate NUM_GENERATIONS * SOLUTIONS_PER_GENERATION random solutions, saving the best from each generation. We use random generation to generate SOLUTIONS_PER_GENERATION purely random solutions. We save the best one, and do this NUM_GENERATIONS times. By the end, we will have NUM_GENERATIONS "good" solutions. This could be used as a good seed for other algorithms that start with an initial population.
