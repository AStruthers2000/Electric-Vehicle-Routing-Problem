Detailed Description

The Node structure consists of an x and y coordinate, the value of the demand at this node (always 0 for charging nodes), a bool that represents if this node is a charger or not (depot and charging nodes both have demand = 0, so we need more specificity), and the index of this node in the list of all nodes. The depot is always index = 0.
